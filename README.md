# EAAI-115358 Supplementary Materials

This repository provides the MATLAB source code, Excel result tables, and figures supporting the accepted article:

**"Genetic Algorithm for Bi-Objective Optimization under Min-Product Fuzzy Relation Inequality Constraints in Supply Chains"**

The article has been accepted by *Engineering Applications of Artificial Intelligence*.

## Description

These supplementary materials support the computational experiments reported in the article. The study investigates a structure-aware genetic algorithm for solving bi-objective optimization problems under min-product fuzzy relation inequality constraints in supply chain settings.

The materials include MATLAB source code, Excel result tables, figures, and related computational result files generated from the proposed algorithm and comparison algorithms. They can be used to verify, interpret, and reproduce the computational results presented in the paper.

## Files

All supplementary materials are provided in the compressed archive:

[`EAAI_115358_MATLAB.zip`](./EAAI_115358_MATLAB.zip)

The archive is organized into seven folders, from `test1` to `test7`. Each folder corresponds to a numerical test instance and contains the MATLAB scripts, result tables, figures, and related result files for that instance.

## Steps to Reproduce

1. Download and extract `EAAI_115358_MATLAB.zip`.
2. Open the folder for a specific numerical test instance, such as `test1`, `test2`, ..., or `test7`.
3. Run the MATLAB scripts provided in that folder.
4. Compare the generated outputs with the Excel tables and figures included in the same folder.
