# EAAI-115358-supplementary-materials
# Code and data for "Genetic Algorithm for Bi-Objective Optimization under Min-Product Fuzzy Relation Inequality Constraints in Supply Chains"

This repository contains the MATLAB source code, Excel result tables, and figures supporting the article:

"Genetic Algorithm for Bi-Objective Optimization under Min-Product Fuzzy Relation Inequality Constraints in Supply Chains"

The article has been accepted by Engineering Applications of Artificial Intelligence.

## Description

This dataset supports the computational experiments reported in the article. The research investigates whether a structure-aware genetic algorithm can effectively solve bi-objective optimization problems under min-product fuzzy relation inequality constraints in supply chain settings.

The repository contains MATLAB source code, Excel result tables, and figures. The data were generated by running the proposed algorithm and comparison algorithms on the numerical test instances described in the article. The Excel tables and figures report the resulting performance indicators and can be used to verify, interpret, and reproduce the computational results presented in the paper.

## Files

The materials are provided in the compressed file:

EAAI_115358.zip

The archive is organized into seven folders, from test1 to test7. Each folder contains the MATLAB code, result tables, and figures for the corresponding numerical test instance.

## Steps to reproduce

To use the dataset, open the folder of a specific test instance and run the MATLAB scripts provided in that folder. The result tables and figures in the same folder correspond to the computational results reported in the article.
